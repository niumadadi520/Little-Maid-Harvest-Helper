## 中文说明

**小女仆丰收助手**是一个适用于 Minecraft 1.20.1 Forge 的东方小女仆非官方兼容模组。

东方小女仆原有的农场模式主要通过破坏并补种的方式收获作物，无法正确处理甜浆果、发光浆果，以及其他模组中需要成熟后右键采摘的作物。

本模组为农场模式增加右键采摘能力。女仆可以寻找附近的成熟果实，主动移动到作物旁并执行右键收获，同时保留能够重复结果的植株。

### 功能

- 支持原版甜浆果和发光浆果。
- 支持其他模组添加的右键采摘作物。
- 支持地面作物、果实灌木和悬挂藤蔓作物。
- 支持注册名包含 `berry`、`berries`、`melon`、`fruit`、`apple`、`grape`、`banana`、`currant`、`bush`、`vine` 等关键词的模组作物。
- 识别 `age`、`stage`、`berries`、`ripe`、`mature`、`fruiting` 等成熟属性。
- 女仆处于农场模式时会扫描附近的成熟果实。
- 女仆能够主动前往作物附近并执行右键收获。
- 忽略手持锄头触发的破坏模式，避免毁坏可重复采摘的植株。
- 将右键后生成的地面掉落物转移到女仆库存。
- 将右键后直接进入交互者背包的产物转移到女仆库存。

### 安装方法

1. 安装 Minecraft 1.20.1 Forge。
2. 安装东方小女仆及其所需依赖。
3. 下载本模组的 Jar 文件。
4. 将 Jar 文件放入客户端和服务端的 `mods` 目录。
5. 启动游戏，将女仆切换到农场模式。

请勿同时安装本模组的多个版本。

### 工作范围

女仆处于农场模式时，本模组会定期扫描女仆周围的成熟果实作物。

发现目标后，女仆会主动接近作物。当距离足够近时，女仆会调用作物自身的右键交互逻辑进行收获，而不是直接破坏植株。

这套机制可以处理原版发光浆果等悬挂作物，也能处理部分无法被东方小女仆原有农场寻路识别的非标准作物。
